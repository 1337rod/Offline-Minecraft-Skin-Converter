# Offline Minecraft Skin Converter

A tool that converts your custom skin into a resource pack, working 100% offline in the browser.

### How it Works

The tool processes your skin file locally in your browser without sending any data to a server. It automatically creates a **`.zip` file in the correct resource pack format**, which replaces **all 9 default Minecraft skins** with your custom skin.

### How to Use

1. Drag and drop your 64x64 skin file.
2. Click "Download .zip" and place the file in your game's `resourcepacks` folder.

<br>

You can access the website [here](https://1337rod.github.io/offline-minecraft-skin-converter/)!
