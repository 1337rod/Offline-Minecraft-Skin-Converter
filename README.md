# Offline Minecraft Skin Converter

---

## 🇧🇷 Português

Uma ferramenta para criar um pacote de recursos do Minecraft com sua skin, funcionando 100% no navegador.

### ✨ Recursos

- **Processamento Offline:** Todo o processo de conversão da skin acontece no seu navegador.
- **100% Seguro:** Não há envio de dados para servidores.
- **Design Simples:** Interface intuitiva de arrastar e soltar.

### ⚙️ Como Usar

1. Arraste e solte seu arquivo PNG de skin (com formato 64x64) para a área indicada.
2. Clique em "Baixar .zip".
3. Ative o pacote de recursos no seu jogo para ver a skin.

A tool to create a Minecraft resource pack with your skin, which works 100% offline in the browser.

### ✨ Features

- **Offline Processing:** All skin conversion happens right in your browser.
- **100% Safe:** No data is sent to any servers.
- **Simple Design:** Intuitive drag-and-drop interface.

### ⚙️ How to Use

1. Drag and drop your skin PNG file (64x64 format) into the designated area.
2. Click "Download .zip".
3. Activate the resource pack in your game to see your skin.
