# Contributing to Offline Minecraft Skin Converter

Thank you for your interest in contributing to the Offline Minecraft Skin Converter! We're excited to have you on board. By participating in this project, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md).

## How Can I Contribute?

### Report Bugs

If you find a bug, please check the [issues page](https://github.com/1337rod/Offline-Minecraft-Skin-Converter/issues) to see if it has already been reported. If not, open a new issue and provide a clear, concise description of the problem, including steps to reproduce it.

### Suggest New Features

We're always open to new ideas! If you have a feature you'd like to see, please open a new issue and describe the functionality and the benefits it would bring to the project.

### Contribute Code

If you're ready to contribute code, follow these simple steps:

1.  **Fork** the repository and clone it to your local machine.
2.  Create a new branch with a descriptive name (e.g., `feature/add-new-language` or `bugfix/fix-download-issue`).
3.  Make your changes, ensuring the code is well-formatted and adheres to the existing style.
4.  Write clear and concise commit messages.
5.  **Push** your branch to your forked repository.
6.  Open a **Pull Request** to the `main` branch of this repository. Provide a detailed description of your changes.
