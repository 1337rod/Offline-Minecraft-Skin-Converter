# Security Policy

## Reporting a Vulnerability

The Offline Minecraft Skin Converter is designed to be a secure, client-side application. It does not send any user data or images to a server, as all processing is done locally within your web browser.

However, if you believe you have found a security vulnerability in the project, we encourage you to report it to us as soon as possible.

### How to Report

Please report security vulnerabilities by opening a new private issue on GitHub. To do this, please send an email to the repository owner at contatoeuurod@gmail.com.

When reporting, please provide the following information:

* A brief description of the vulnerability.
* Steps to reproduce the vulnerability.
* The version of the project you were using.
* Any additional context or information that could be helpful.

We will review your report and work to address the vulnerability in a timely manner. We appreciate your efforts in helping to keep this project secure.
